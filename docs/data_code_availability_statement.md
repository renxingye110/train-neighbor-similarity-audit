# Data and code availability statement

## Suggested manuscript text

Code, processed result tables, figure source data, and supporting reproducibility files for this study are available at https://github.com/renxingye110/train-neighbor-similarity-audit. Raw third-party peptide and protein sequence records should be obtained from the original public databases cited in the manuscript and are not redistributed here. The repository includes the derived tables used for the main figures, supplementary tables, and the retrospective prioritization analysis. A frozen public release of the repository is archived at https://doi.org/10.5281/zenodo.21235886.
