# Third-party data sources

This repository does not redistribute raw third-party database records. The analyses reported in the manuscript draw on public sources that should be accessed from their original homes and cited as described in the manuscript.

## Datasets used directly in the reported results

- ESCAPE benchmark (Harvard Dataverse DOI: 10.7910/DVN/C69MCD; original GitHub record)
- APD6
- DRAMP / DRAMP_antibacterial
- DBAASP
- DeepLoc 2.0 benchmark
- AMPCliff benchmark from Li et al.
- LL-37 truncation data from the published experimental source cited in the supplement

## Additional sources used during local preprocessing and quality control

- ChEMBL
- DRAVP
- CancerPPD
- DRAMP activity-specific subsets
- Human-fragment negatives assembled locally for preprocessing

## Redistribution policy in this repository

- Raw source records are not copied here.
- Derived figure tables and machine-readable manuscript tables are included because they support the reported results.
- Any reuse of derived tables remains subject to the terms of the underlying source databases where those terms apply.
